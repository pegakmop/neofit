# neofit
Данный проект создан для облегчения соблюдения личной конфиденцальности пользователя при использовании entware с помощью sing-box-go пакета предоставляя веб интерфейс и настройку на роутере за пользователя одной командой. Проект представлен для в познавательных целях, автор проекта не несет ответственности за нарушение пользователем законов его страны и за неправомерное использование данного софта пользователем.

Установить можно командой:
```
terminal
```
